# Utkal Dental Care — Premium Concept Redesign

A premium, modern concept redesign for **Utkal Dental Care**, a clinic specializing in dental, skin, and hair treatments based in Bhubaneswar, Odisha.

This repository features the **Atelier** design system layout — utilizing warm cream tones, high-contrast typography, and soft-shadow component widgets to deliver a professional, clean, and patient-focused web experience.

---

## 🎨 Visual Identity & Color System

- **Primary Background (`cream` / `bone`):** `#F3F0E9` (A warm, soothing bone/cream background)
- **Surface Cards (`paper`):** `#FBF9F3` (Soft white canvas for cards and details)
- **Text & Headings (`ink`):** `#15130F` (Near-black warm ink for high contrast and readability)
- **Brand Accent (`clay`):** `#E4572E` (Vibrant vermilion for CTAs, step badges, and highlights)
- **Borders & Separators (`line`):** `#DED7C7` (Soft grey-brown border accents)

---

## ✨ Features

- **Responsive Grid System:** Pixel-perfect layouts responsive across mobile (375px), tablet (768px), and desktop (1240px+) screen widths.
- **Stats Dashboard Widgets:** Clean, interactive numeric counters styled as premium cards with hover translations.
- **Visit Timeline Steps:** A structured, vertical-to-horizontal sequence card block detailing a patient's first clinic visit step by step.
- **Interactive Before/After Slider:** Drag-enabled component to showcase dental, skin, or hair treatment transformations.
- **Scroll Reveal Animations:** Micro-interactions that fade and slide elements on scroll for an engaging user experience.
- **Unified Navigation:** Custom sticky header and structured footer with contact details, business hours, and site pages.

---

## 🛠️ Technology Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS (Utility classes & CSS variables)
- **Icons:** Tabler Icons React (`@tabler/icons-react`)
- **Compilation/Tooling:** PostCSS, TypeScript, Vitest

## 🚀 Getting Started

### Prerequisites

Make sure you have Node.js (v18+) and npm installed.

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the local development server:
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) in your browser to view the site.

3. Compile the production static build:
   ```bash
   npm run build
   ```
